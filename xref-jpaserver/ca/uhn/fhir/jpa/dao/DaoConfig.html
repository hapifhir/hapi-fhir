168 	 * If set to <code>true</code> (default is <code>false</code>) the server will allow
169 	 * resources to have references to external servers. For example if this server is
170 	 * running at <code>http://example.com/fhir</code> and this setting is set to
171 	 * <code>true</code> the server will allow a Patient resource to be saved with a
172 	 * Patient.organization value of <code>http://foo.com/Organization/1</code>.
173 	 * <p>
174 	 * Under the default behaviour if this value has not been changed, the above
175 	 * resource would be rejected by the server because it requires all references
176 	 * to be resolvable on the local server.
177 	 * </p>
178 	 * <p>
179 	 * Note that external references will be indexed by the server and may be searched
180 	 * (e.g. <code>Patient:organization</code>), but
181 	 * chained searches (e.g. <code>Patient:organization.name</code>) will not work across 
182 	 * these references.
183 	 * </p>
184 	 * <p>
185 	 * It is recommended to also set {@link #setTreatBaseUrlsAsLocal(Set)} if this value
186 	 * is set to <code>true</code>

397 	/**
398 	 * This setting may be used to advise the server that any references found in
399 	 * resources that have any of the base URLs given here will be replaced with
400 	 * simple local references.
401 	 * <p>
402 	 * For example, if the set contains the value <code>http://example.com/base/</code>;
403 	 * and a resource is submitted to the server that contains a reference to
404 	 * <code>http://example.com/base/Patient/1</code>, the server will automatically
405 	 * convert this reference to <code>Patient/1</code>
406 	 * </p>
407 	 * 
408 	 * @param theTreatBaseUrlsAsLocal The set of base URLs. May be <code>null</code>, which
409 	 * means no references will be treated as external
410 	 */
411 	public void setTreatBaseUrlsAsLocal(Set<String> theTreatBaseUrlsAsLocal) {
412 		HashSet<String> treatBaseUrlsAsLocal = new HashSet<String>();
413 		for (String next : ObjectUtils.defaultIfNull(theTreatBaseUrlsAsLocal, new HashSet<String>())) {
414 			while (next.endsWith("/")) {
415 				next = next.substring(0, next.length() - 1);
416 			}
417 			treatBaseUrlsAsLocal.add(next);
418 		}
419 		myTreatBaseUrlsAsLocal = treatBaseUrlsAsLocal;
420 	}
