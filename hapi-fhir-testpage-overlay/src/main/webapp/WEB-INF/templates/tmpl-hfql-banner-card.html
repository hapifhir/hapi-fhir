HFQL / SQL Execution

This page can be used to execute queries using the HAPI FHIR Query Language (HFQL), which is a SQL-like syntax for querying FHIR repositories. Learn more about HFQL architecture at https://smilecdr.com/docs/hfql/, and learn more about the HFQL syntax at https://smilecdr.com/docs/hfql/sql_syntax.html. This UI will display a maximum of [[${rowLimit}]] rows.
